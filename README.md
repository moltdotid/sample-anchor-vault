# Sample Anchor Vault

A simple Solana vault program built with Anchor framework.

## Features

- **Initialize**: Create a vault PDA for an authority
- **Deposit**: Transfer SPL tokens into the vault
- **Withdraw**: Authority can withdraw tokens from the vault

## Structure

```
programs/
  sample-vault/
    src/
      lib.rs       # Main program logic
    Cargo.toml     # Dependencies
```

## Instructions

### Initialize
Creates a new vault PDA for the signer.

### Deposit
Transfers tokens from user's token account to vault's token account.

### Withdraw
Authority can transfer tokens from vault back to their account using PDA signing.

## Security Features

- PDA-based vault with authority control
- Bump seed stored to prevent reinitialization
- Has-one constraint on withdraw to ensure only authority can withdraw

## Build & Deploy

```bash
anchor build
anchor deploy
```
